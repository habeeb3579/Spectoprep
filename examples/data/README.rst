Corn NIR benchmark data
=======================

Place the public Cargill / Eigenvector ``corn.mat`` here as ``corn.mat``.

Download: https://eigenvector.com/data/Corn/

Fallback path used by scripts/notebooks: ``/tmp/spectoprep-data/corn.mat``.
Override with the ``SPECTOPREP_CORN_MAT`` environment variable.
